## 🎯 CASOS DE USO PRINCIPALES

1. **Registrar Compra**
   - Ingresa factura con productos
   - Sistema actualiza inventario
   - Registra en Kardex automáticamente

2. **Crear Receta**
   - Selecciona ingredientes
   - Sistema calcula costo
   - Muestra margen en tiempo real

3. **Procesar Producción**
   - Elige receta y cantidad
   - Sistema descuenta inventario
   - Registra salida en Kardex
   - Agrega a producto terminado

4. **Consultar Kardex**
   - Ver todos los movimientos
   - Filtrar por producto
   - Exportar a Excel

5. **Exportar Reportes**
   - Recetas con costos actualizados
   - Inventario completo
   - Kardex detallado
